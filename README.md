# Social Media Full Stack

Una aplicación completa de red social desarrollada con React y Spring Boot, inspirada en tutoriales modernos de desarrollo full stack.

## Estructura del Proyecto

- **frontend/** - Aplicación React con arquitectura modular y componentes reutilizables
- **backend/** - API RESTful desarrollada con Spring Boot siguiendo buenas prácticas

## Tecnologías Utilizadas

### Frontend
- **React**: Biblioteca de JavaScript para interfaces de usuario
- **Redux**: Para gestión avanzada del estado de la aplicación
- **Tailwind CSS**: Framework de utilidades CSS para diseño personalizado
- **Material UI**: Componentes React pre-diseñados
- **Formik**: Manejo de formularios con validación
- **React Router**: Navegación entre páginas
- **WebSocket**: Para comunicación en tiempo real (chat)

### Backend
- **Spring Boot**: Framework Java para aplicaciones empresariales
- **Spring Security**: Gestión de autenticación y autorización
- **Spring Data JPA**: Capa de persistencia simplificada
- **Hibernate**: ORM para mapeo objeto-relacional
- **MySQL**: Base de datos relacional
- **JWT**: Autenticación basada en tokens
- **WebSocket**: Comunicación bidireccional en tiempo real

## Características Principales

- **Autenticación Completa**: Registro, inicio de sesión, recuperación de contraseña
- **Feed Personalizado**: Visualización de publicaciones de perfiles seguidos
- **Publicaciones**: Creación, edición y eliminación de posts con imágenes
- **Sistema de Perfiles**: Información de usuario personalizable con foto de perfil
- **Interacciones**: Me gusta, comentarios y compartir contenido
- **Chat en Tiempo Real**: Comunicación instantánea entre usuarios
- **Exploración**: Descubrimiento de contenido y usuarios nuevos
- **Modo Oscuro/Claro**: Soporte para cambio de tema

## Arquitectura

El proyecto sigue una arquitectura de microservicios:

- **Capa de Presentación**: Frontend React con componentes modulares
- **Capa de API**: Backend Spring con controladores REST
- **Capa de Servicios**: Lógica de negocio encapsulada
- **Capa de Datos**: Repositorios y modelos para persistencia

## Instalación y Ejecución

### Frontend
```
cd frontend
npm install
npm start
```

### Backend
```
cd backend
./mvnw spring-boot:run
```
